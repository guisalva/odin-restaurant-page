# Odin - Restaurant page

![Status Badge](https://img.shields.io/badge/status-completed-brightgreen?style=for-the-badge)

This project was built to practice **ES6 Modules** and **Webpack bundler** as a part of the JavaScript curriculum from [The Odin Project](https://www.theodinproject.com).

It is a simple bakery website featuring tab navigation between different sections. All page content is dynamically generated using JavaScript modules and DOM manipulation.

## 🛠️ Tech Stack

- HTML
- CSS
- JS
- Webpack

## 🚀 Installation and Usage

Clone the repository:

```bash
git clone https://github.com/guisalva/odin-restaurant-page.git

cd odin-restaurant-page
```

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npx webpack serve
```
